Another question about the same sum: Limit of a sequence defined by a sum: [imath]\lim_{n\to \infty} \frac1{2^n}\sum_{k=1}^n \frac1{\sqrt k}\binom nk[/imath]. Found using Approach0.

Let [imath]a_k = k^{-1/2}[/imath]. Notice that [imath](a_k)[/imath] decreases to [imath]0[/imath]. Then for each fixed [imath]m \geq 1[/imath] and for all [imath]n \geq m[/imath],
[imath] \frac{1}{2^n} \sum_{k=1}^{n} \binom{n}{k} a_k \leq \frac{1}{2^n} \underbrace{\sum_{k=1}^{m} \binom{n}{k} (a_k - a_m)}_{= \mathcal{O}(n^m)} + \frac{1}{2^n} \underbrace{\sum_{k=1}^{n} \binom{n}{k} a_m}_{=(2^n - 1)a_m}. [/imath]
Taking limsup as [imath]n\to\infty[/imath], it follows that
[imath] \limsup_{n\to\infty} \frac{1}{2^n} \sum_{k=1}^{n} \binom{n}{k} a_k \leq a_m [/imath]
Since [imath]a_m \to 0[/imath] as [imath]m\to\infty[/imath], this proves
[imath] \lim_{n\to\infty} \frac{1}{2^n} \sum_{k=1}^{n} \binom{n}{k} a_k = 0. [/imath]
Addendum. I just saw that OP is a high-school student. Here is a little tweak of the argument above that does not use any fancy analysis stuffs.
Let [imath]m_n = \lfloor \log n \rfloor[/imath]. Then for [imath]n \geq 3[/imath], we always have [imath]1 \leq m_n \leq n[/imath]. Then
\begin{align*}
0
\leq \frac{1}{2^n} \sum_{k=1}^{n} \binom{n}{k} \frac{1}{\sqrt{k}}
&= \frac{1}{2^n} \sum_{k=1}^{m_n} \binom{n}{k} \frac{1}{\sqrt{k}}
+ \frac{1}{2^n} \sum_{k=m_n + 1}^{n} \binom{n}{k} \frac{1}{\sqrt{k}} \\
&\leq \frac{1}{2^n} \sum_{k=1}^{m_n} n^k
+ \frac{1}{2^n} \sum_{k=m_n + 1}^{n} \binom{n}{k} \frac{1}{\sqrt{m_n}} \tag{1} \\
&\leq \frac{n^{1+m_n}}{2^n}
+ \frac{1}{\sqrt{m_n}}. \tag{2}
\end{align*}
For [imath]\text{(1)}[/imath] I utilized the fact that [imath]\binom{n}{k} \leq n^k[/imath] and [imath]\frac{1}{\sqrt{k}}[/imath] is decreasing. For [imath]\text{(2)}[/imath] I utilized the geometric sum formula and the identity [imath]\sum_{k=0}^{n} \binom{n}{k} = 2^n[/imath].
Now taking [imath]n\to\infty[/imath] and applying the squeezing lemma proves the claim.

[imath]\lim_{n \to \infty} \frac {1}{2^n} \sum_{k=1}^n \frac{{n}\choose{k}}{\sqrt{k}}=0[/imath]
I offer a sketch proof. The basic idea is as follows: for large [imath]n[/imath], the small values of [imath]k[/imath] will make small contributions, because [imath]\frac{{n \choose k}}{2^n}[/imath] will be small there, and the large values of [imath]k[/imath] will make small contributions because [imath]\sqrt{\frac{1}{k}}[/imath] will be small there. Thus, the whole sum should be small too.
Thus, as desired, the limit is [imath]0[/imath].

It is enough to apply Laplace's method.
Through the inverse Laplace transform we have [imath]\frac{1}{\sqrt{k}}=\int_{0}^{+\infty}\frac{e^{-ks}}{\sqrt{\pi s}}\,ds [/imath], hence
[imath] \sum_{k=1}^{n}\binom{n}{k}\frac{1}{\sqrt{k}} = \int_{0}^{+\infty}\frac{-1+(1+e^{-s})^n}{\sqrt{\pi s}}\,ds =\frac{2}{\sqrt{\pi}}\int_{0}^{+\infty}\left[(1+e^{-s^2})^n-1\right]\,ds[/imath]
where the last integrand function behaves like [imath](2^n-1) e^{-ns^2/2}[/imath].
It follows that
[imath] \sum_{k=1}^{n}\binom{n}{k}\frac{1}{\sqrt{k}} \approx (2^n-1)\sqrt{\frac{2}{n}} [/imath]
and the wanted limit is simply zero.

Using Abel's summation and the fact that [imath]\sum \limits_{k=1}^{n} \binom{n}{k} = 2^{n}-1[/imath] you can solve the above question.
From the Abel's summation we can say that [imath]\sum \limits_{k=1}^{n} \frac{\binom{n}{k}}{\sqrt{k}} \approx \frac{\sum \limits_{k=1}^{n} \binom{n}{k}}{\sqrt{n}}-\int \limits_{1}^{n} ( \sum \limits_{k=1}^{t} \binom{t}{k} \frac{d}{dx}(\sqrt{t})) dt [/imath]
By substituting we arrive at : [imath] \frac{2^{n}-1}{\sqrt{n}}-\int \limits_{1}^{n} ( 2^{t}-1) \frac{d}{dx}(\sqrt{t}) dt [/imath]
It easy by comparison tests to bound the integral part by [imath]\frac{2^n}{\sqrt{n}}[/imath].
So its at most between [imath]0[/imath] and [imath]2\frac{2^n}{\sqrt{n}}[/imath] which both approach [imath]0[/imath] when divided by [imath]2^n[/imath] when [imath]n \to \infty[/imath].

The limit is part of a broad class of limits for which the classical analysis designed Toeplitz theorem. The theorem says that given an array of real numbers, [imath]\lambda_{n,k}: 1\le k\le n,\ n\ge1 [/imath], such that: [imath]i) \ \lim_{n\to\infty}\lambda_{n,k}=0, \ \forall k\in \mathbb{N}[/imath], [imath]ii) \lim_{n\to\infty} \sum_{k=1}^n \lambda_{n,k}=1[/imath] and [imath]iii)[/imath] there exists [imath]C>0[/imath] such that for all positive integers [imath]n[/imath], [imath]\sum_{k=1}^n |\lambda_{n,k}|\le C[/imath], then for any convergent sequence [imath]\delta_k[/imath], we have
[imath]\lim_{n\to\infty} \sum_{k=1}^n \lambda_{n,k} \delta_k=\lim_{n\to\infty}\delta_n .[/imath]
In your problem set [imath]\displaystyle \lambda_{n,k}=\frac{1}{2^n}\binom{n}{k}[/imath] and [imath]\displaystyle \delta_k=\frac{1}{\sqrt{k}}[/imath], and since the conditions are satisfied, we have
[imath]\displaystyle \lim_{n \to \infty} \frac {1}{2^n} \displaystyle \sum_{k=1}^n \frac{\displaystyle \binom{n}{k}}{\sqrt{k}}=\lim_{n\to\infty} \frac{1}{\sqrt{n}}=0,[/imath]
which ends the little proof.
