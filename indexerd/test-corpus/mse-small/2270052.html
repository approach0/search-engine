If [imath]3x^2\equiv1\bmod p[/imath] then 3 is a quadratic residue

Unsure of how this implication has occurred. I know that if 3 is a QR then there exists [imath]x^2\equiv3\bmod p[/imath]. However, how does this become [imath]3x^2\equiv1\bmod p[/imath]?

Well, [imath]x^2\equiv 3^{-1}\pmod p\implies x^{-2}\equiv 3 \pmod p[/imath]
Multiplication by [imath]3[/imath].
@franzlemmermeyer Right, that's about it...

Since [imath]\gcd(3,p) = 1[/imath], 3 has a unique multiplicative inverse modulo [imath]p[/imath], say, it is [imath]3^{-1}[/imath]. Then original congruence becomes
[imath] 3^{-1} \cdot 3 x^2 \equiv 3^{-1} \bmod p.[/imath]
So [imath]3^{-1}[/imath] is a quadratic residue. Then apply
[imath] 1 = \left( \frac{1}p\right) = \left( \frac{3}p\right) \left( \frac{3^{-1}}p\right) [/imath]
to get [imath]\left( \frac{3}p\right) = 1[/imath], i.e., 3 is a qudratic residue modulo [imath]p[/imath].
